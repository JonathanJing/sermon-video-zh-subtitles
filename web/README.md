# Web Prototype

This is a frontend-only operator console prototype for the Mariners Church Chinese subtitle pipeline.

Open `index.html` directly in a browser, or serve the `web/` folder with any static server.

Prototype scope:

- Live source monitoring flow for 8:30 PT first, 10:00 PT fallback.
- Realtime Chinese caption workspace.
- Scripture sidebar, glossary, notes placeholder.
- Review timeline controls and VTT/SRT export buttons.
